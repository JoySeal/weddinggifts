[Article on Codrops](https://tympanus.net/codrops/?p=52157)
[Demo](http://tympanus.net/Development/HorizontalSmoothScrollLayout/)


## Installation

Install dependencies:

```
npm install
```

Compile the code for development and start a local server:

```
npm start
```

Create the build:

```
npm run build
```
